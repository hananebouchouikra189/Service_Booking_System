# Service Booking System

This project is a Service Booking System built using Angular, Spring Boot, and MySQL.

## Features

- **Company Features:**
  - Create, manage, and approve bookings.
  - Post and manage ads.
  
- **Client Features:**
  - Search for services.
  - Book services.
  - Review services.

## Technologies Used

- **Frontend:** Angular
- **Backend:** Spring Boot
- **Database:** MySQL
