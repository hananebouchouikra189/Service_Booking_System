# Service Booking System

This project is a Service Booking System built using Angular, Spring Boot, and MySQL.

## Features

- **Company Features:**
  -Create Account
  -Login
  -Post Ads
  -Update & Delete Ads
  -View Bookings
  -Approve Or Reject Booking
  
- **Client Features:**
  -Create Account
  -Login
  -Search Services
  -View Ad Details and Reviews
  -Book Services
  -View Bookings
  -Review Services

## Technologies Used

- **Frontend:** Angular
- **Backend:** Spring Boot
- **Database:** MySQL
- **Style:** Ng Zorro UI
